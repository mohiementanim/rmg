480 Interrelationship Digraph (Network Diagram) If you’re like me and you thought the spelling of Digraph was an error, it’s not. The difference between a digraph and a diagram is that a digraph is a graph emphasizing direction. A diagram can be any image outlining a plan or how something works. However, many articles on the web relate Interrelationship Digraph and Interrelationship Diagrams as the same thing. Other names for this tool include Relations Diagram, Network Diagram, or Matrix Diagram. The Interrelationship Digraph is a 7M tool. It often uses input from other tools – like a Fishbone Diagram or an Affinity Diagram to define drivers and outcomes in a process.
