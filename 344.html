343 ● No target(s) (or not knowing that there is a target) ● Wrong target(s) ● Too many targets Fuzzy Targets What if the target you were shooting at was really “fuzzy” and it was hard to see where you are aiming? This is a common problem in most organizations. They have KPIs that can be interpreted differently by different people. For example, “On-time Delivery” can be measured from different origination points and different endpoints. Because we don’t truly know the target, we aim at where we think the target is.
