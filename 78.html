77 eVent Fabric What is eVent fabric? eVent is a waterproof fabric made by eVent Fabrics. As an expanded polytetrafluorethylene (ePTFE) fabric, eVent is in direct competition with GORE-TEX, the legally acknowledged inventor of ePTFE. While eVent is highly similar to GORE-TEX in many ways, this fabric features proprietary Direct Venting technology, which increases breathability and reduces the possibility of overheating.
