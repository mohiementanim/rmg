524 Predefined Process Shows a named process which is defined elsewhere. Represented as a rectangle with double-struck vertical edges. On-page Connector Pairs of labeled connectors replace long or confusing lines on a flowchart page. Represented by a small circle with a letter inside. Off-page Connector A labeled connector for use when the target is on another page. Represented as a home plate-shaped pentagon. Other symbols The ANSI/ISO standards include symbols beyond the basic shapes. Some are: Shape Name Description Data File or Database Data represented by a cylinder symbolizing a disk drive. Document Single documents represented as a rectangle with a wavy base. Multiple documents represented as a stack of rectangles with wavy bases.
